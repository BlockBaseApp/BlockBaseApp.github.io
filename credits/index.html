Credits and inspiration

Inspired by the launchers Minecraft players already trust.

BlockBaseMC is an independent custom launcher. It is not affiliated with Mojang, Microsoft, Prism Launcher, CurseForge, Modrinth, MultiMC, or any other launcher, but it learns from the workflows those apps made familiar.

Prism Launcher

Inspired the instance-first workflow, isolated folders, clean profile management, import thinking, and practical power-user controls.

CurseForge App

Inspired the idea of browsing modpacks and content from inside the launcher instead of manually moving files around.

Modrinth App

Inspired a modern content-first experience with clean project browsing, versions, loaders, and fast installs.

MultiMC

Inspired the value of simple instance folders, transparent launch settings, and launcher behavior that stays close to Minecraft's files.

Mojang and Microsoft

Minecraft metadata, assets, libraries, account ownership, and Java profile data are provided through official Minecraft and Microsoft services.

Open source tools

Electron, Node.js, electron-builder, and the JavaScript ecosystem make the desktop app and portable Windows build possible.

Important note

Names like Minecraft, Mojang, Microsoft, CurseForge, Modrinth, Prism Launcher, and MultiMC belong to their owners. They are mentioned here only to explain compatibility, APIs, credits, and design inspiration.
