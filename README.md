# BlockBaseMC Website

This repository serves the official BlockBaseMC website at:

```text
https://blockbaseapp.github.io/
```

It is a static GitHub Pages site. The launcher source lives separately in:

```text
https://github.com/itz-me-fisherYT/BlockBase-Launcher
```

The download buttons point to the launcher's latest GitHub Release. The site also includes an Updates page that tracks the build history through the current local build.
