What it is

A custom launcher built around practical Minecraft sessions.

The goal is not to clone another launcher. BlockBaseMC takes ideas players expect, like isolated instances, account switching, modpack imports, and quick repairs, then wraps them in a cleaner custom interface.

1

Profiles first

Create isolated Java instances with their own version, loader, RAM, args, game folder, and account.

2

Accounts included

Use Microsoft Java login, reauth saved sessions, switch accounts, or add offline/dev accounts for testing.

3

Mods and packs

Browse Modrinth and CurseForge, install projects into the selected profile, and manage installed content.

Why it exists

Built because launching Minecraft should feel less painful.

BlockBaseMC is made for custom packs, server hopping, quick testing, and keeping Minecraft folders readable. It focuses on Java Edition now so the core launch path can get stable before expanding further.

  • Import existing Prism, MultiMC, ZIP, Modrinth, CurseForge, and .minecraft setups.
  • Use browser-based Microsoft login instead of a tiny broken webview.
  • Keep the launcher responsive while downloads and game processes run.
  • Show logs, crash summaries, profile folders, and running instance controls.
