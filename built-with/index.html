How it was built

An Electron app with a custom Java launcher backend.

BlockBaseMC is a desktop app. The interface runs in Electron, while the main process handles Minecraft downloads, Microsoft authentication, file imports, process launching, and operating-system actions.

Desktop shell

Electron

Electron provides the Windows desktop window, native dialogs, secure preload API, browser login handoff, and portable executable packaging.

Frontend

HTML, CSS, and JavaScript

The UI is a static app enhanced with custom JavaScript for accounts, imports, server cards, mod browsing, content management, and profile tools.

Launcher backend

Node.js services

The Electron main process downloads Minecraft metadata, assets, libraries, loader files, extracts natives, starts Java, and streams logs back to the UI.

Build output

electron-builder

Portable Windows builds are produced with electron-builder. The current local build path is release-fast57.

Services and APIs

The launcher talks to the same ecosystem Minecraft players already use.

MS

Microsoft and Minecraft auth

Microsoft browser/device login, Xbox Live auth, XSTS auth, Minecraft Services login, entitlement checks, and profile lookup.

MR

Modrinth

Search and install mods, modpacks, shaders, and resource packs from Modrinth project and version metadata.

CF

CurseForge

Search and install supported CurseForge projects through the CurseForge API when a user supplies an API key.
